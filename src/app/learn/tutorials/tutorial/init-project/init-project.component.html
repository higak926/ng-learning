新規プロジェクト作成

では、プロジェクトを作成していきましょう！
ワークスペース(デスクトップなど)でCLIを使ってAngularプロジェクトを作成します。

ng new angular-tour-of-heroes
① Angularの初期設定

ルーティング設定とstylesheetを選択します

② 新規作成プロジェクトの確認

initializeが成功するとangular-tour-of-heroes(指定した名称)ディレクトリが作成されます。

③ Angular 起動！

作成したプロジェクトからAngularを起動してみましょう♪

④ Welcomページの確認

Welcomページが表示されたら大成功です🎉🎉
