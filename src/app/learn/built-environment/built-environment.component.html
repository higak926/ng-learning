(噂の)環境構築 〜簡単3ステップ〜
Step 1. まずはnode.js！

Angularの環境を構築するために、ますはnodeをインストールしましょう。
無事インストールできると、下記コマンドでバージョンを確認することができます

node.js 公式サイト node --version
v16.15.1
# バージョン出なかった人はインストール失敗しているかも！
# 頑張ってインストールしましょう 注1.) Angularは多くの機能や関数をnpmパッケージに依存しているため、node.jsをインストールしてnpmパッケージを利用できるようにします 注2.) node.jsのバージョンはお好みですが、あまり新しすぎるとかえって動かない(ググったときの情報が少ない)可能性があるため、LTSをお勧めします！
Step 2. お次にAngular CLI！

Angularプロジェクトの作成やコンポーネントの生成、テスト、デプロイなど様々な開発タスクのお役に立つAngular CLIをインストールしましょう。

注1.) Step 1でnode.jsをインストールしているため、npmコマンドが使えるようになっていると思います npm install -g @angular/cli@13.3.9
# -gでグローバルインストール、@~でバージョン指定できるよ！ 注2.) 無事インストールできたら、下記コマンドでバージョン確認してみましょう ng v
イカしたAngularロゴとcliやdevkitなどのバージョン情報が表示されたら、無事インストール完了！
Step 3. Angularプロジェクトの作成

Step 2でインストールしたAngular CLIを使って、プロジェクトを作成しましょう！

注1.) ng newコマンドでプロジェクト名を指定して作成することができます ng new my-app
# my-appという名前でプロジェクト作成 注2.) プロジェクトが作成できたら、ローカルサーバーを起動して動作確認してみましょう cd my-app
ng serve --open
# http://localhost:4200/にアクセスしてサーバーが立ち上がっていれば、無事動作確認は完了！
# control + Cでサーバを止めることができるよ

ここまで確認できれば、一先ずAngular開発できる準備は完了です
